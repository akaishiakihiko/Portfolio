自己紹介

こんにちは！ 北海道はお寿司がおいしい街”小樽”出身の赤石暁彦です。 今回、サイテクカレッジ美浜校webスペシャリスト科にて3カ月間、webサイト制作のコーディングについて学ばせていただきました。学んだ内容を惜しみなく、制作に表現いたしましたので、どうぞごゆっくりご覧ください。

制作
第１回制作

名桜大学

[ 制作背景 ]
「教育」をテーマ。名桜大学の元々の濃い色使いでギュッとしたデザインから、桜のやわらかさ・学生のフレッシュさを表現した淡い色使いと、空間を広くとったわかりやすく見やすいレイアウトを工夫しました。

[ ポイント ]
HTML・cssを学習後の初制作。トップページとコンテンツ２ページ制作。XDでワイヤー・デザインフレーム作成後、デザインの書き出し、HTML・cssでのコーディング。タグの名前決めや、レイアウトの整理を工夫しました。

[ 使用ソフト ] HTML / css / XD / Photoshop

第2回制作

ウミカジ カフェ

[ 制作背景 ]
「飲食業」がテーマ。豊見城市ウミカジテラス10周年記念を祝しOPENする「レストランカフェ」をテーマに企画・制作しました。メニューでは美味しそうな写真を俯瞰で画面いっぱいに、１つの縦長の長いサイトとして工夫し制作しました。

[ ポイント ]
2回目の制作では、学習したscssをより活かせるようwidthは指定せず全画面でのコーディングを意識し、レスポンシブデザインも学びコーディングに取り入れました。游明朝フォントを使用し綺麗ですっきりとした表現にしました。

[ 使用ソフト ] HTML / scss / jQuery / XD / Photoshop / レスポンシブ

第3回制作

ちゅらトマト

[ 制作背景 ]
前回制作したレストランの続編として、レストランメニューのパスタやピザに使用した、「豊見城産ちゅらトマト」のwebサイトを企画・制作しました。トマトのフレッシュさや、つくる時のこだわりを表現しました。

[ ポイント ]
トマトのポップで明るいの色合い、游ゴシックフォントで見やすいイメージを表現しました。学習したｊQueryでは、フェードスライド・ループスライド・フェードアップ・ポップアップ等、10種類を使用しました。

[ 使用ソフト ] HTML / scss / jQuery / XD / Photoshop / レスポンシブ

第3回制作

ウミカジ クッキー

[ 制作背景 ]
前回制作したレストランの続編として、レストランで食事されたお客様のお土産として、「Umikaji cookie」を企画しました。cafeが親サイト、トマトとクッキーがコンテンツページになるように意識しました。

[ ポイント ]
クッキーのクールでシックの色合い、游明朝ですっきりとしたイメージを表現しました。学習したｊQueryを10種類使用し、トマトとのイメージの対比になるよう、同レイアウトのイメージの違いがわかるサイトとなりました。

[ 使用ソフト ] HTML / scss / jQuery / XD / Photoshop / レスポンシブ
